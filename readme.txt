Appends the post author's display name to posts. Useful for multi-author blogs that are crossposted elsewhere, such as LiveJournal or Facebook.

== Description ==

This plugin appends the author's display name to posts and/or pages. There is an option to turn off the display on your own site, so the signature is visible only in feeds and crossposted content.

== Installation ==

1. Upload the plugin directory to `/wp-content/plugins/` 
1. Activate the plugin through the 'Plugins' menu in WordPress
